# The dataset contains 50 features describing hospital encounter of diabetes patients

## Features groups include:

### Demographics
- `race`
- `gender`
- `age`

### Hospital utilization 
- `time_in_hospital`
- `num_lab_procedures`
- `num_medications`
- `number_outpatient`
- `number_emergency`
- `number_inpatient`

### Administrative features
- `admission_type_id`
- `discharge_disposition_id`
- `admission_source_id`

### Diagnosis codes
- `diag_1`
- `diag_2`
- `daig_3`

### Medication features
- `insulin`
- `metformin`
- `glyburide`
- `others`
