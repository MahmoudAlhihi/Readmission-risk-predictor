# Readmission Risk Predictor
An end-to-end healthcare machine learning system that predicts 30-day hospital readmission risk using real-world clincal data.

## Goals
- Train a baseline model
- Evaluate using ROC-AUC, precision/recall
- Expose model via FastAPI 
- Deploy to AWS
- Document full pipeline

## Project Structure
- src/ -> ML pipeline
- notebooks/ -> EDA + experiments
- api/ -> inference API
- docs/ -> technical documentation
- models/ -> trained artifcats
